# Annotation Notes for Igbo NLP Tasks

These notes summarize basic annotation ideas used in language datasets.

## Common NLP Tasks

1. Text Classification  
Classifying text into categories such as news, education, or politics.

2. Natural Language Inference (NLI)  
Determining whether one sentence logically follows another.

3. Named Entity Recognition  
Identifying names of people, locations, or organizations in text.

Example (Igbo)

Sentence:
Gọvanọ gara Enugwu taa.

Translation:
The Governor went to Enugu today.

Entities:
Person: Gọvanọ  
Location: Enugu
